# RELATORIO DE ENTREGA DO T2

## Introducao 
Aqui consta o relotorio de entrega para o trabalho dois da disciplina de verificação e validação de software, foi entregue um roteiro de testes em java produzido com
a extensão Selenium IDE para o firefox em cima do sistema de teste do produto Orange HRM.

## Roteiro
O roteiro contempla o seguinte fluxo de usuario: Login -> aba Time -> Project Info - (Project, Customer) -> Customer -> Add Customer -> Logout

## Como Executar
-Crie um codespaces
-Va na aba de extensões
-Verifique se a extensao java for vscode esta instalada
-Abra o terminal

cd t2seleniumverival/
mvn test
